主要使用的第三方框架
==============

1.  `pod 'STTwitter'`.
2.  `pod 'FMDB`.
3.  `pod 'MJPhotoBrowser`.
4.  `pod 'LSPlayer'`.
5.  `pod 'AFNetworking`.
6.  `pod 'SDWebImage`.
